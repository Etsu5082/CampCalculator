---

## 2. 画面一覧

| No | 画面ID | 画面名 | 概要 |
|----|--------|--------|------|
| 1 | LOGIN | ログイン画面 | パスワード認証 |
| 2 | DASHBOARD | ダッシュボード | 合宿一覧、新規作成 |
| 3 | CAMP_NEW | 合宿新規作成 | 基本情報入力 |
| 4 | CAMP_DETAIL | 合宿詳細 | タブ切り替えで各種設定 |
| 5 | RESULT | 計算結果 | 精算金額一覧 |
| 6 | PARTIAL_SCHEDULE | 途参途抜一覧 | 途中参加・途中抜けのスケジュール表 |
| 7 | EXPORT | 出力確認 | PDF/Excel出力 |

## 3. 各画面詳細

### 3.1 ログイン画面 (LOGIN)

```
┌────────────────────────────────────────┐
│                                        │
│        合宿費用計算アプリ              │
│                                        │
│    ┌─────────────────────────────┐    │
│    │ パスワード: [**********   ] │    │
│    └─────────────────────────────┘    │
│                                        │
│           [ ログイン ]                 │
│                                        │
└────────────────────────────────────────┘
```

**機能**
- パスワード入力
- 認証成功 → ダッシュボードへ
- 認証失敗 → エラーメッセージ表示

**編集可能な項目**:
- 合宿名
- コート1面あたり料金
- 1泊料金、保険料、1日目昼食対象/対象外
- 食事単価（朝食/昼食/夕食の追加・削除価格）
- バス代（往復一括または往路/復路別設定）
- 高速代（往路/復路）
- レンタカーの有効/無効と料金設定

**注意**: 日程（開始日/終了日/泊数）は参加者・スケジュールに影響するため編集不可。変更が必要な場合は合宿を新規作成する。

**時間割UIの操作方法**:
- 各マスをクリックすると、そのスロット（日×時間帯）の参加者を対象とした雑費追加モーダルが開く
- 既存の雑費は該当するマスに表示され、クリックで編集可能
- 建て替え者が設定されている場合は [名前] で表示
- 全員対象の雑費は別途一覧で管理

**建て替え総額表示**:
- 建て替え者が設定されている雑費の総額を人ごとに集計
- 内訳として各費目と金額を表示
- 精算時の参考情報として利用
