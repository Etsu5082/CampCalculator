## 3. 各画面詳細

### 3.1 ログイン画面 (LOGIN)

```
┌────────────────────────────────────────┐
│                                        │
│        合宿費用計算アプリ              │
│                                        │
│    ┌─────────────────────────────┐    │
│    │ パスワード: [**********   ] │    │
│    └─────────────────────────────┘    │
│                                        │
│           [ ログイン ]                 │
│                                        │
└────────────────────────────────────────┘
```

**機能**
- パスワード入力
- 認証成功 → ダッシュボードへ
- 認証失敗 → エラーメッセージ表示

---

### 3.2 ダッシュボード (DASHBOARD)

```
┌────────────────────────────────────────────────────────────┐
│ 合宿費用計算アプリ                        [ ログアウト ]   │
├────────────────────────────────────────────────────────────┤
│                                                            │
│  [ + 新規合宿作成 ]                                        │
│                                                            │
│  ┌────────────────────────────────────────────────────┐   │
│  │ 合宿一覧                                            │   │
│  ├────────────────────────────────────────────────────┤   │
│  │ 📁 2024年度春合宿    2024/03/20-23   3泊4日  [複製][詳細]│ │
│  │ 📁 2023年度秋合宿    2023/09/15-18   3泊4日  [複製][詳細]│ │
│  │ 📁 2023年度春合宿    2023/03/18-21   3泊4日  [複製][詳細]│ │
│  └──────────────────────────────────────────────────────┘   │
│                                                            │
└────────────────────────────────────────────────────────────┘
```

**機能**
- 過去の合宿一覧表示（新しい順）
- 新規合宿作成ボタン
- **過去の合宿を複製ボタン**（日程設定・費用設定をコピー、参加者はコピーしない）
- 各合宿の詳細画面へのリンク
- **使い方ガイド**（折りたたみ式、初めてのユーザー向け）

**使い方ガイドの内容**
1. 合宿を作成する
2. 参加者を登録する（CSV一括登録の説明含む）
3. 途中参加・途中抜けを設定する
4. 日程を設定する（オプション）
5. 雑費を登録する（オプション）
6. 計算結果を確認・出力する
- よくある質問（食事計算、端数処理など）

**編集可能な項目**:
- 合宿名
- コート1面あたり料金
- 1泊料金、保険料、1日目昼食対象/対象外
- 食事単価（朝食/昼食/夕食の追加・削除価格）

- 高速代（往路/復路）
- レンタカーの有効/無効と料金設定

**注意**: 日程（開始日/終了日/泊数）は参加者・スケジュールに影響するため編集不可。変更が必要な場合は合宿を新規作成する。

5. **その他のPDF（AI-OCR対応）**
   - スキャンPDFや未対応フォーマットはClaude Vision APIでOCR解析
   - 画像として認識し、料金情報をAIが抽出

**エラーハンドリング**
- PDF読み取り失敗 → エラーメッセージ表示、再アップロード促す
- 未対応フォーマット → 警告表示、手動入力を促す
- 必須項目抽出失敗 → 警告表示、編集画面で手動入力可能
