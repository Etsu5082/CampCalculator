## 5. 発見された問題点と修正事項

### ⚠️ 問題1: pending.phpでpending APIを呼び出していない

**ファイル**: `src/views/members/pending.php`
**現在**: `fetch('/index.php?route=api/members/pending')`
**問題**: パスが正しいが、ルーティングのパターンマッチを確認する必要あり
**確認必要**: Routerが `/api/members/pending` を `/api/members/{id}` より優先してマッチするか

### 解決策: ルート順序を確認

Routerは登録順にマッチングを試みるため、`/api/members/pending` は `/api/members/{id}` より **前** に登録されている必要があります。

**index.phpの現在の順序**:
```php
Line 108: $router->get('/api/members', ...);
Line 109: $router->post('/api/members', ...);
Line 110: $router->get('/api/members/{id}', ...);      // ← これより前に pending を配置する必要がある
Line 113: $router->get('/api/members/pending', ...);   // ← 問題！{id}より後ろ
```

**修正必要**: `/api/members/pending` を `/api/members/{id}` より **前** に移動
